## Implementation notes

Use `dotnet new` templates:

```powershell
dotnet new classlib -n Moza.ScLink.Core -o src/Moza.ScLink.Core -f net8.0
dotnet new classlib -n Moza.ScLink.DirectInput -o src/Moza.ScLink.DirectInput -f net8.0-windows
# ... etc
dotnet new wpf -n Moza.ScLink.App -o src/Moza.ScLink.App
dotnet new xunit -n Moza.ScLink.Core.Tests -o tests/Moza.ScLink.Core.Tests
# ... etc
```

Then create the solution and add all projects:

```powershell
dotnet new sln -n MozaStarCitizenLink
Get-ChildItem -Recurse -Filter "*.csproj" | ForEach-Object { dotnet sln add $_.FullName }
```

For the WPF project's TFM, edit the generated `.csproj` to use `net8.0-windows10.0.19041.0` so Windows.Graphics.Capture interop is available in later phases.

Delete the default `Class1.cs` from each class library before committing.
