## Implementation notes

For `ImmutableDictionary` usage, prefer `ImmutableDictionary<string, double>.Empty` as the default value and provide a constructor overload accepting `IEnumerable<KeyValuePair<string, double>>` for ergonomic construction.

`ForceCommand` is an abstract record. Subtypes use positional record syntax:

```csharp
public sealed record PlayEffectCommand(
    ForceEffect Effect,
    double FinalIntensity) : ForceCommand;
```

The base `ForceCommand` carries `CommandId` and `IssuedAt` so all commands have correlation IDs for diagnostics.
